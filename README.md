## Tools for development

- React JS
- Node JS
- HTML
- CSS
- MySQL

## Execution commands

```
- Backend
  - cd D:/PhladRian
  - npm install
  and then
  - npm start
  or
  - pm2 start backend/server.js --exp-backoff-restart-delay=100
- Frontend
  - cd D:/PhladRian/frontend
  - npm install
  - npm start
```
